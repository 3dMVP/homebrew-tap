# Blobit Homebrew Tap

Install Blobit CLI with Homebrew:

```bash
brew tap 3dMVP/tap
brew install blobit
```

Install on Linux or macOS without Homebrew:

```bash
curl -fsSL https://raw.githubusercontent.com/3dMVP/homebrew-tap/main/install.sh | sh
```

The CLI talks to `https://api.blobit.ai` by default.
